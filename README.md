# Audio Visual Analyzer

ブラウザで音声ファイル・動画ファイルを読み込み、再生しながらビジュアライザーを表示するシンプルな Web アプリです。

## 現在の実装内容

### Phase 1 (基本機能)
- 音声 / 動画ファイルの読込
- 再生 / 一時停止 / 停止
- 棒グラフ型アナライザー表示
- 色相・色相幅・輝度・彩度の調整
- 感度・スムージング・棒幅の調整
- 表示比率の切替（16:9 / 1:1）

### Phase 2 (表現拡張)
- **5種類のアナライザータイプ**: 棒グラフ / 波形線 / 点 / 円形放射 / ミラー対称
- **0dB基準位置切替**: 底辺基準 / 中央基準（上下対称）
- **レイヤー1〜4対応**: 音域を自動分割し複数レイヤーを重ね描き
- **レイヤー個別設定**: 各レイヤーの色相オフセット・感度を独立調整

## セットアップ

本プロジェクトは HTML / CSS / JavaScript のみで構成されており、`npm install` などは不要です。

### 方法1: そのまま開く

1. このリポジトリをクローンします。
2. `index.html` を Chrome または Edge で開きます。

### 方法2: ローカルサーバーで開く

直接開けない環境では、リポジトリ直下で簡易サーバーを起動してください。

```bash
python3 -m http.server 8000
```

起動後、ブラウザで `http://localhost:8000` を開きます。

## 使いかた

1. `ファイルを選択` を押して、音声ファイルまたは動画ファイルを読み込みます。
2. `再生` を押すと、ビジュアライザー表示が始まります。
3. `表現タイプ` でアナライザーの種類を切り替えます。
4. `0dB基準` で底辺 / 中央を切り替えます。
5. `レイヤー数` で1〜4を選択し、多音域表示を試します。
   - 2レイヤー以上で各レイヤーの色相オフセット・感度を個別に調整できます。
6. `色調整` / `感度・形状` で全体パラメーターを調整します。

## 動作確認環境

- Google Chrome
- Microsoft Edge

## 補足

- 対応するファイル形式はブラウザ依存です。
- 初回再生時はブラウザの制限によりユーザー操作が必要です。
- 詳細仕様は `doc/spec.md` を参照してください。

# Audio Visual Analyzer

ブラウザで音声ファイル・動画ファイルを読み込み、再生しながらビジュアライザーを表示するシンプルな Web アプリです。

## デモ

https://youtube.com/shorts/dywUYpVFatw?feature=share

## セットアップ

本プロジェクトは HTML / CSS / JavaScript のみで構成されており、`npm install` などは不要です。

### 方法1: そのまま開く

1. このリポジトリをクローンします。
2. `index.html` を Chrome または Edge で開きます。

### 方法2: ローカルサーバーで開く

直接開けない環境では、リポジトリ直下で簡易サーバーを起動してください。

```bash
python3 -m http.server 8000
```

起動後、ブラウザで `http://localhost:8000` を開きます。

同じ Wi-Fi 上のスマホなどからテストする場合は、`localhost` の代わりに PC の
IP アドレス（Mac / Linux は `ifconfig`、Windows は `ipconfig` で確認）を使って
`http://<PCのIPアドレス>:8000` を開きます。

## 使いかた

1. `ファイルを選択` を押して、音声ファイルまたは動画ファイルを読み込みます。
2. `再生` を押すと、ビジュアライザー表示が始まります。
3. **アナライザー** セクションで設定を変更します。
   - `タイプ` で棒グラフ / 円形放射を切り替えます。
   - `表現方法` で棒 / 波形線 / 点を選択します。
   - 棒グラフ時は `表示モード` で通常 / ミラー（上下）/ ミラー（左右）を選べます。
   - `アナライザーランダム` でタイプ・表現方法・表示モード・レイヤー数・レイヤー色相オフセットを一括ランダム設定します。
   - `レイヤー数` で1〜4を選択し、多音域表示を試します（2レイヤー以上で各レイヤーの色相オフセット・感度を個別調整可能）。
4. **色調整** セクションで色を変更します。
   - `色相ランダム` ボタンで色相とレイヤーオフセットをランダマイズします。
   - `色相連続変化` をONにすると、再生中に色相が自動変化します。
5. **感度・形状** セクションで表現を調整します。
   - `密度` で描画要素の数を調整します。
   - `基準点オフセット` で0dB基準位置をずらします。
   - `残像強度` でトレイル効果の強さを変更します。
   - `形状ランダム` で感度・スムージング・線の太さ・密度・基準点オフセット・残像強度を一括ランダム設定します。
6. **録画する場合**は、`録画モード` に切り替えます。
   - `FPS` で録画フレームレート（25 / 29.97 / 30）を選択できます。
   - `録画開始` を押すと再生と同時に録画が始まります。
   - `録画停止` で録画を終了し、`保存` で録画MIMEに応じた動画ファイル（`.mp4` / `.webm`）をダウンロードします。
   - `再録画` で録画をリセットしてやり直せます。

## スマホでの利用について

モバイル最適化は正式な対象外（`doc/spec.md` §2.3）ですが、動作確認は可能です。
以下の点にご注意ください。

- **初回再生**: モバイルの自動再生制限のため、必ず画面を操作して `再生` を押してください。
- **レイアウト**: 本体はデスクトップの2カラム前提のため、スマホでは操作パネルが窮屈に表示されます。
- **録画**: Android Chrome は動作見込みです。iOS Safari は `MediaRecorder` の
  挙動差（出力形式・ビットレート指定の扱い）があり、録画品質は PC と同等になりません。

## 現在の実装内容

### Phase 1 (基本機能)
- 音声 / 動画ファイルの読込
- 再生 / 一時停止 / 停止
- 棒グラフ型アナライザー表示
- 色相・色相幅・輝度・彩度の調整
- 感度・スムージング・棒幅の調整
- 表示比率の切替（16:9 / 1:1）

### Phase 2 (表現拡張)
- 複数アナライザータイプ対応
- 0dB基準位置切替
- レイヤー1〜4対応（音域自動分割）
- レイヤー個別設定（色相オフセット・感度）

### Phase 3 (表現拡張・追加仕様)
- **アナライザータイプ切替**: 棒グラフ / 円形放射
- **表現方法切替**: 棒 / 波形線 / 点
- **棒グラフ表示モード**: 通常 / ミラー（上下）/ ミラー（左右）
- **密度調整**: 描画要素数を 30〜100 の範囲で調整（最小でも表示が消えない）
- **基準点オフセット**: 0dB基準位置を最大 99 まで移動
- **線の太さ調整**: 描画要素の太さを変更
- **色相ランダマイズ**: 色相とレイヤーごとの色相オフセットをランダム設定
- **アナライザーランダマイズ**: タイプ・表現方法・表示モード・レイヤー数・レイヤー色相オフセットをワンクリックでランダム設定
- **形状ランダマイズ**: 感度・スムージング・線の太さ・密度・基準点オフセット・残像強度をワンクリックでランダム設定
- **色相連続変化モード**: 再生中に色相が自動的に変化し続ける
- **残像表現**: 残像強度を調整し、描画にトレイル効果を付与（過剰設定は上限でクランプ）
- **解析帯域**: 50Hz〜15kHz に限定（超低域・超高域ノイズを除外）
- **レイヤー設定**: アナライザーセクション内に統合

### Phase 6 (拡張表現)
既存の棒グラフ / 円形放射に加え、13種類のアナライザータイプを追加（計15種類）。タイプは「タイプ」セレクトで系統別に選択できます。

- **時間軸系**: スペクトログラム（滝）/ 円形スペクトログラム / 3D地形 / トンネル
- **擬似3D系**: 擬似3Dバー / 回転3Dリング
- **流体・粒子系**: パーティクル放出 / 波紋 / ノイズフロー / メタボール
- **幾何系**: オシロスコープ（リサージュ）/ 極座標フラワー / ボロノイ脈動
- **追加パラメーター**（選択タイプに応じて表示）: 時間幅（履歴長）/ 動きの速さ / 要素量 / 粘性揺らぎ
- 選択タイプが対応しないコントロール（表現方法・表示モード・レイヤー等）は自動的に隠れます。
- 外部ライブラリなし・Canvas 2D のみで実装（擬似3Dは平行投影、メタボールは blur+contrast 合成）。

### 録画機能
- **モード切替**: 再生モード / 録画モード
- **録画**: ビジュアライザー描画領域を音声付きで録画（MP4優先、非対応環境はWebMへフォールバック）
- **MP4互換性**: MP4対応環境では H.264/AAC の一般的なプロファイル候補を優先して選択
- **A/V同期**: AudioContext の再開と録画キャプチャの開始を待ってから再生を始めることで、冒頭の映像・音声のずれを抑制
- **エンコード品質**: 解像度・FPSに応じて映像ビットレートを自動算出（6〜24Mbps）、音声は192kbps
- **WebM長さ情報**: WebM 出力時はヘッダーに再生時間（Duration）を書き込み、編集ソフトで長さ・シークが正しく扱えるファイルとして保存
- **FPS指定**: 25fps / 29.97fps / 30fps を選択可能
- **操作**: 録画開始 / 停止 / 保存 / 再録画
- **自動命名**: 日時ベースのファイル名で保存（拡張子は `.mp4` / `.webm` を自動判定）
